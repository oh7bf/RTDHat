# RTDHat
Raspberry Pi HAT for RTDs.

Files
=====

```textile
RTDHat-B_Cu.gbl        - bottom Copper layer
RTDHat-B_Mask.gbs      - bottom solder mask
RTDHat-Edge_Cuts.gm1   - edge cuts
RTDHat-F_Cu.gtl        - top Copper layer
RTDHat-F_Mask.gts      - top solder mask
RTDHat-F_Paste.gtp     - top solder paste
RTDHat-F_SilkS.gto     - top silk screen
RTDHat-NPTH.drl        - drill file for non-plated through holes
RTDHat-PTH.drl         - drill file for plated through holes
RTDHat.pdf             - circuit diagram 
```
